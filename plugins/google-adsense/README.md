# google-adsense

A plugin for Blessing Skin Server that can let you display Google Ads with Google AdSense in the website.

## Usage

Enable this plugin.

Copy your client-id (ca-pub-xxxxxxxxxxxxxx) and save it in the settings of the plugin.
